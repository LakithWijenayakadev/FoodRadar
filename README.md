# FoodRadar
React Native app that shows a list of nearby restaurants based on the user’s location. 
=======
## Restaurant App (FoodRadar)

This is a simple React Native mobile application (Android/IOS) that shows nearby restaurants based on the user's current location. The app uses Google Maps and location APIs to fetch and display real restaurant data.

## Features

* Detects the user's current location
* Displays nearby restaurants on the map
* The user can search for restaurants and get directions by clicking the direction button
* Shows a list of restaurants sorted by distance or rating
* Simple restaurant card UI with menu, distance, ratings, and direction button to get directions with maps or ride application (Uber)
* Automatically search nearby locations by default

## Tech Stack

* React Native (CLI)
* Google Maps API
* Geolocation API

## Setup

1. Clone the repository

git clone https://github.com/your-username/Resturant_App.git
cd Resturant_App

2. Install dependencies

npm install

3. Start Metro

npm start

4. Run the app

npm run android
